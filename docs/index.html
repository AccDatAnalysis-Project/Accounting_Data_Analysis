会計情報のデータ分析

会計情報のデータ分析

最終更新日：2020/04/12

　近年，会計データを用いた経営分析は，単なる比率の分析だけでなく，計量経済学の知見を取り入れて発展してきた。本書は，このような統計的な技法を含めた会計データ分析の入門書である。このような分析に必要となるのは，

  • 会計学の知識
  • 統計学の知識
  • 統計ソフトの知識

である。本書は，会計学の知識については，日商簿記検定の3級程度，統計学や統計ソフトの知識はほとんどないという学部生を対象として執筆された。入門書であるが，データの整形作業などのデータ分析の必要な工程は最低限網羅している。本書の演習を通じて，会計データ分析の世界へ第一歩を生み出してほしい。
